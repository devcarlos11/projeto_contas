# Controle Financeiro - Projeto Java Spring MVC

Este é um projeto Java Spring MVC desenvolvido para permitir aos usuários gerenciar suas contas a pagar e a receber. Facilita o controle financeiro pessoal ou empresarial, fornecendo recursos para registrar transações financeiras, visualizar saldos e gerar relatórios simples.

## Requisitos

Antes de executar o projeto, certifique-se de ter os seguintes requisitos instalados:

1. **Apache Tomcat 9:**
   - Baixe o Apache Tomcat 9 [aqui](https://tomcat.apache.org/download-90.cgi).
   - Siga as instruções de instalação fornecidas na documentação oficial.
